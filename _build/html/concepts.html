Concepts¶

There are few things that you need to understand about how Tago works in order to optimize your applications. Watch this short video to learn about the bucket/device/token concepts.



Bucket¶

Bucket is where the data from your devices are stored. You may create as many buckets as you wish.

To create a bucket, click on Data on the sidebar. Then, click on Add new bucket blue button.

Enter with the bucket name and a short description. You can edit these fields later if needed.

Under the General Information session, you can find detailed information about the bucket. Select the bucket, click twice on it, and you will see this screen:

Visible option¶

If you need to make any reference to a bucket, like when you access it from a script in the Analysis, you will need to use the bucket id. The bucket id can be found under the bucket general information. This is an unique number generated by Tago system, and cannot be modified.
